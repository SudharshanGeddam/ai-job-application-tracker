---
title: "Get Application"
api: "GET /api/applications/{id}"
description: "Fetch a single job application by ID."
authMethod: "bearer"
---

## Path Parameters

<ParamField path="id" type="string" required>
  The UUID of the application.
</ParamField>

<RequestExample>
```bash curl
curl http://localhost:3000/api/applications/uuid-here \
  -H "Authorization: Bearer <token>"
```
</RequestExample>
