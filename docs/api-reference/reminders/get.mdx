---
title: "Get Reminder"
api: "GET /api/reminders/{id}"
description: "Fetch a single reminder by ID."
authMethod: "bearer"
---

## Path Parameters

<ParamField path="id" type="string" required>UUID of the reminder.</ParamField>

<RequestExample>
```bash curl
curl http://localhost:3000/api/reminders/reminder-uuid \
  -H "Authorization: Bearer <token>"
```
</RequestExample>

<ResponseExample>
```json 200 Success
{
  "success": true,
  "data": {
    "id": "reminder-uuid",
    "applicationId": "app-uuid",
    "message": "Follow up on application",
    "remindAt": "2026-07-01T09:00:00.000Z",
    "isCompleted": false,
    "application": {
      "id": "app-uuid",
      "companyName": "Acme Corp",
      "role": "Senior Engineer",
      "status": "APPLIED"
    }
  }
}
```
```json 403 Forbidden
{
  "success": false,
  "status": "fail",
  "message": "You do not have permission to access this reminder"
}
```
</ResponseExample>
