---
title: "List Reminders"
api: "GET /api/reminders"
description: "Fetch all reminders for the authenticated user, sorted by scheduled date ascending."
authMethod: "bearer"
---

<RequestExample>
```bash curl
curl http://localhost:3000/api/reminders \
  -H "Authorization: Bearer <token>"
```
</RequestExample>
