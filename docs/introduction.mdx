## What is AI Job Tracker?

AI Job Tracker is a backend API that helps job seekers **organize, track, and supercharge** their job search. It goes beyond a simple spreadsheet by combining a structured application tracker with a suite of AI features powered by **Groq (Llama 3.3-70B)**.

## Key Features

<AccordionGroup>
  <Accordion icon="lock" title="JWT Authentication">
    Secure register/login flow with token-based auth to keep user data private and safe.
  </Accordion>
  <Accordion icon="file-pdf" title="Resume Upload & Extraction">
    PDF upload via Cloudinary with automatic text extraction to power AI features seamlessly.
  </Accordion>
  <Accordion icon="chart-mixed" title="AI JD Match Score">
    AI-scored gap analysis between your resume and a job description to identify missing skills.
  </Accordion>
  <Accordion icon="envelope-open-text" title="Smart Cover Letters">
    Two-stage LLM pipeline: talking points → polished draft based on verified resume data.
  </Accordion>
  <Accordion icon="calendar-clock" title="Automated Reminders & Jobs">
    Schedule and receive email reminders tied to specific applications. Background jobs auto-mark stale applications after 30 days of inactivity.
  </Accordion>
</AccordionGroup>

## Tech Stack

| Layer | Technology |
|---|---|
| Runtime | Node.js (Express 5) |
| Database | PostgreSQL via Prisma ORM |
| AI | Groq (Llama 3.3-70B) |
| File Storage | Cloudinary |
| Email | Nodemailer (Gmail) |
| Validation | Zod |
| Deployment | Docker + docker-compose |

## API at a Glance

| Group | Endpoints |
|---|---|
| Auth | `POST /api/auth/register`, `POST /api/auth/login`, `GET /api/auth/me` |
| Applications | `GET /POST /PUT /DELETE /api/applications` + stats |
| AI | `POST /:id/analyze-jd`, `POST /:id/cover-letter`, `POST /:id/interview-tips` |
| Reminders | `GET /POST /PATCH /DELETE /api/reminders` |

<Note>
  All API responses follow a consistent envelope: `{ success: boolean, data: any, message?: string }`.
</Note>
