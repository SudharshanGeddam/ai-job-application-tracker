## Prerequisites

Make sure you have the following installed:

- **Node.js** v18+ and npm
- **PostgreSQL** (local or cloud — [Neon](https://neon.tech) is free)
- **Docker** (optional, for the containerized setup)

You'll also need accounts and API keys from:

## Step 1: Clone the Repository

```bash
git clone https://github.com/SudharshanGeddam/ai-job-application-tracker.git
cd ai-job-application-tracker
```

## Step 2: Install Dependencies

```bash
npm install
```

## Step 3: Configure Environment Variables

Copy the example file and fill in your values:

```bash
cp .env.example .env
```

Open `.env` and fill in your credentials:

```bash .env
PORT=3000
NODE_ENV=development

# Your PostgreSQL connection string
DATABASE_URL="postgresql://postgres:password@localhost:5432/job_tracker_db?schema=public"

# Generate with: node -e "console.log(require('crypto').randomBytes(64).toString('hex'))"
JWT_SECRET="your-long-random-secret"
JWT_EXPIRES_IN="7d"

# From cloudinary.com dashboard
CLOUDINARY_CLOUD_NAME="your-cloud-name"
CLOUDINARY_API_KEY="your-api-key"
CLOUDINARY_API_SECRET="your-api-secret"

# From console.groq.com
GROQ_API_KEY="gsk_..."

# Gmail + App Password
EMAIL_USER="your-email@gmail.com"
EMAIL_PASSWORD="your-app-password"
```

## Step 4: Run Database Migrations

```bash
npx prisma migrate dev
```

This creates all the tables (`User`, `Application`, `Reminder`) in your database.

## Step 5: Start the Server

```bash
npm run dev
```

You should see:

```
[Server] Running on http://localhost:3000 in development mode
[CRON] Stale application job scheduled
[CRON] Reminder job scheduled
```

## Step 6: Test the Health Endpoint

```bash
curl http://localhost:3000/health
```

```json
{
  "status": "ok",
  "uptime": 1.234,
  "timestamp": "2026-06-25T00:00:00.000Z"
}
```

## Step 7: Register Your First User

```bash
curl -X POST http://localhost:3000/api/auth/register \
  -H "Content-Type: application/json" \
  -d '{"name": "Jane Doe", "email": "jane@example.com", "password": "securepass123"}'
```

```json
{
  "success": true,
  "message": "Account created successfully.",
  "data": {
    "id": "uuid-here",
    "name": "Jane Doe",
    "email": "jane@example.com",
    "createdAt": "2026-06-25T00:00:00.000Z"
  }
}
```

## Docker Alternative

Prefer containers? Use docker-compose to spin up both the app and a Postgres database:

```bash
docker-compose up --build
```

See the [Docker Guide](/guides/docker) for full details.
